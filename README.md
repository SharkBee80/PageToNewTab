# PageToNewTab

## 项目概述
PageToNewTab 是一个浏览器扩展程序，旨在改善用户的网页浏览体验。该扩展允许用户将指定网页设置为新标签页。

## 使用场景
一些新标签页只有网页版本,而浏览器设置无法在新建标签页`(Ctrl+T)`时自动打开自定义网页。

## 功能特性
- **自动转移**：根据预设规则自动将特定类型的链接在新标签页中打开。
- **自定义设置**：自定义设置打开的网页。

## 安装说明
1. 下载并解压项目文件。
2. 打开浏览器的扩展管理页面（通常位于 `chrome://extensions/`）。
3. 启用“开发者模式”。
4. 选择“加载已解压的扩展程序”，然后导航到项目目录进行加载。

## 使用方法
1. 安装完成后，点击浏览器工具栏中的 PageToNewTab 图标以访问设置界面。
2. 根据个人需求调整各项参数，例如启用自动转移功能等。
3. 浏览网页时，符合设定条件的链接将会自动在新标签页中打开。
